F – Formal / IF – Informal / S – Slang

F: Hello - สวัสดี

S: Yo / Alright - หวัดดี / ไง
F: Sir - ท่านผู้ชาย
F: Madam / Ma’am - คุณผู้หญิง
IF: Mate / Dude / Buddy / Pal - เพื่อนสนิท
IF: Guys / Gals - พวก / สาวๆ
S: Bro / Sis - เพื่อนสนิทมาก
