"-ed " pronunciation

1. Voicced เสียงก้อง - d :
ถ้าพยัญชนะตัวท้ายเป็นเสียงก้อง cdต้องออกเสียงเป็น d
(พยัญชนะที่อยู่ในประเภท voiced เช่น b,dg,g,j,l,m,n,r,v,w,y,z เช่น " Played " ก็ต้องออกเสียงเหมือนสะกดเป็น " Playd ")
2. Voiccless ไม่มีเสียงก้อง t :
ถ้าพยัญชนะตัวท้ายไม่เป็นเสียงก้อง ed ต้องออกเสียงเป็น t พยัญชนะที่อยู่ในประเภท voiceless
( เช่น c,ch,f,gh(f), k,p,q,s,sh,x อย่างช่น คำว่า Kissed ก็ต้องออกเสียงเหมือนสะกดเป็น Kiss)
3. -t/d=id :
ถ้าพยัญชนะตวท้ายเป็นตัวอักษร d หรือ t คำที่ลงท้ายด้วย คำที่ลงท้ายด้วย ed
(ต้องออกเสียงเป็น id )
