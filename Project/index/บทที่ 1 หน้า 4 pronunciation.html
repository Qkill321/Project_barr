"-ed" Pronunciation

1. Voiced เสียงก้อง - /d/ :
ถ้าพยัญชนะตัวท้ายเป็นเสียงก้อง ed จะออกเสียงเป็น /d/
(ตัวอย่างเสียง: b, dg, g, j, l, m, n, r, v, w, y, z — เช่น "played" → "playd")
2. Voiceless ไม่มีเสียงก้อง - /t/ :
ถ้าพยัญชนะตัวท้ายไม่มีเสียงก้อง ed จะออกเสียงเป็น /t/
(ตัวอย่างเสียง: c, ch, f, gh(f), k, p, q, s, sh, x — เช่น "kissed" → "kisst")
3. ลงท้ายด้วย -t / -d ออกเสียงว่า /ɪd/ :
ถ้าคำลงท้ายด้วย t หรือ d จะออกเสียง ed เป็น /ɪd/
(ตัวอย่าง: "wanted" → "wantɪd", "needed" → "needɪd")
